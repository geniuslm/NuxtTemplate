# Nuxt UI Minimal Starter


## 开始服务器

开始开发服务器在 `http://localhost:3000`:

```bash
# pnpm
pnpm run dev
```
```bash
# pnpm
pnpm run build
```

```bash
# pnpm
pnpm run preview
```
```bash
# pnpm 生成静态网站 到 .output public 
pnpm run generate
```
