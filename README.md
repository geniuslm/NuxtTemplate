# Nuxt UI Minimal Starter


## 开始服务器

开始开发服务器在 `http://localhost:3000`:

```bash
# 开始开发服务器，访问地址：http://localhost:3000
pnpm run dev
```
```bash
# 构建项目
pnpm run build
```

```bash
# 预览构建后的项目
pnpm run preview
```
```bash
# pnpm 生成静态网站 到 .output public 
pnpm run generate
```

## 更新项目
```bash
# 检查过期的包:
pnpm outdated
```
```bash
# 更新所有包到它们的最新版本
pnpm update
```
